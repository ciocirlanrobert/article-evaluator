# News Article NLP

News Article evaluator with NLP.

### Prerequisites

`node` and `npm` are required to run this app.

### Installing

To install all the project dependencies run:
```bash
npm install
```

## Start

Start the app with 
```bash
npm start
```
in your terminal

## Development

Run the app in development mode:

```bash
npm run build-dev
```

## Production

Run the app in production mode:
```bash
npm run build-prod
```
